V2M — Turn Motion Into Music

A personal project transforming movement in space into playable MIDI — live or from recorded video.

Motion becomes melody
Point a camera or drop a clip — V2M translates motion through space into MIDI notes you can record, export, or stream live into your DAW. It’s an instrument made from movement.

About this project

A creative, non-commercial experiment exploring how visual motion maps to musical structure. Not a product — just experiments and demos.

Core tech: Python + YOLOv8 (object detection) + DeepSORT (tracking)
Media & UI: OpenCV for video I/O & FFmpeg for exports; PySide6 for the desktop GUI
MIDI: real-time virtual MIDI port + MIDI file export (mido / python-rtmidi or similar)
Streaming: accepts webcams, RTSP/HTTP & YouTube Live — streams MIDI live to Ableton
How it maps: tracks object trajectories; axis crossings and position → note triggers and velocity mapping
